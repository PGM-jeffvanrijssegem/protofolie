Gentse Feesten

Over dit project

Deze opdracht heeft als intentie een werkplek te simuleren waarin ik gevraagd word om een website te bouwen gebaseerd op een reeds gemaakt design voor een echte klant. Ik zal mijn kennis van HTML, CSS en JS samen moeten brengen om deze website individueel en zelfstandig te kunnen bouwen. Het project maakt gebruik van mijn vaardigheden en kennis om een complete en functionele website te realiseren. Doel: Praktische Toepassing van Vaardigheden: HTML: Structureren van de inhoud van de webpagina's. CSS: Stijlen van de webpagina's volgens het aangeleverde design. JavaScript: Toevoegen van interacties en dynamische elementen aan de webpagina's. Interpretatie van Ontwerpen: Screenshots: Begrijpen en vertalen van visuele ontwerpen naar code. Video: Inzicht krijgen in de interacties en het gedrag van de website op verschillende apparaten (mobiel en desktop). Zelfstandigheid: Ontwikkelen van de website zonder directe begeleiding, wat me voorbereidt op een werkplek waar ik zelfstandig taken moet uitvoeren. Resultaat: Functionele Website: Een complete website die voldoet aan de eisen van de klant, gebaseerd op het aangeleverde design. Responsief Ontwerp: De website moet correct weergegeven worden op zowel desktop als mobiele apparaten, zoals gedemonstreerd in de video. Documentatie en Levering: Het eindproduct moet goed gedocumenteerd zijn zodat anderen de code kunnen begrijpen en onderhouden. Dit kan onder andere bestaan uit commentaar in de code, een README-bestand, en eventueel documentatie voor installatie en gebruik.

Enkele van de belangrijkste kenmerken van dit project zijn:

  • Responsief Ontwerp: Een van de meest cruciale aspecten van moderne webontwikkeling is het zorgen dat de website goed werkt op zowel desktop- als mobiele apparaten.

  • Consistente en Nauwkeurige Implementatie van het Ontwerp: Het is essentieel om het aangeleverde design nauwkeurig om te zetten in een werkende website.

  • Functionaliteit en Interactiviteit: Naast het visuele aspect moet de website ook functioneel en interactief zijn.

Clar development

Over dit project

Als eindopdracht krijg ik een interactief ontwerp aangeleverd van een UI/UX designer van het agency Clar Development. Het is aan mij om deze website zo goed mogelijk na te bouwen op basis van het aangeleverde prototype. De foto's en fonts worden aangeleverd, maar ik moet zelf de pagina's opbouwen en stijlen. Naast de verplichte technische vereisten ben ik vrij en moet ik zelf de HTML-structuur opbouwen, stijlen en animaties toevoegen. Ik zal hiervoor een logisch opgebouwde en overzichtelijke structuur gebruiken. Ik push mijn code bij iedere nieuwe feature, wijziging, sessie, etc. naar mijn GitHub-repo. Het project maakt gebruik van HTML, CSS, JavaScript en GSAP. Doel: Nauwkeurige Reproductie van het Ontwerp: Het pixel-perfect omzetten van het interactieve ontwerp naar een werkende website. Het correct implementeren van alle visuele elementen zoals kleuren, typografie, lay-out en afbeeldingen. Technische Implementatie: Het gebruiken van HTML, CSS en JavaScript om de pagina's te bouwen en te stylen. Het toevoegen van interactieve elementen en animaties om de gebruikerservaring te verbeteren. Het zorgen voor een responsief ontwerp dat goed werkt op zowel desktop als mobiele apparaten. Best Practices en Codekwaliteit: Het opzetten van een logisch opgebouwde en overzichtelijke code-structuur. Het regelmatig pushen van code naar mijn GitHub-repo voor versiebeheer en samenwerking. Het documenteren van de code en het gebruik van duidelijke commit-berichten. Resultaten: Volledig Functionele Website: Een complete en werkende website die nauw aansluit bij het aangeleverde interactieve ontwerp. Correct werkende navigatie, interactieve elementen en animaties die bijdragen aan een positieve gebruikerservaring. Responsief en Toegankelijk Ontwerp: Een website die goed functioneert op verschillende apparaten en schermgroottes. Toegankelijkheid voor alle gebruikers, inclusief die met beperkingen, door het volgen van best practices zoals semantische HTML en ARIA-labels. Geoptimaliseerde en Goed Gedocumenteerde Code: Goed gestructureerde HTML, CSS en JavaScript code die gemakkelijk te begrijpen en te onderhouden is. Een GitHub-repo met regelmatige commits, duidelijke commit-berichten en een gedocumenteerd README-bestand.

Enkele van de belangrijkste kenmerken van dit project zijn:

  • Pixel-Perfect Reproductie: Het is essentieel om het interactieve ontwerp dat door de UI/UX designer is aangeleverd nauwkeurig om te zetten in een werkende website.
  • Responsief Ontwerp: De website moet goed functioneren en er goed uitzien op verschillende apparaten en schermgroottes.
  • Versiebeheer en Documentatie: Het regelmatig pushen van code naar GitHub en het documenteren van de voortgang is cruciaal voor versiebeheer en samenwerking.
